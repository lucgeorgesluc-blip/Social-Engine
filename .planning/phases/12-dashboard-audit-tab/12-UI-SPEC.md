## Interaction Contract

### Tab Switching

```
Trigger: click sidebar "Audit SEO" button
Action: $store.nav.tab = 'audit'
Effect: audit tab content appears (x-show), all other tabs hidden
Data load: auditPanel().init() fetches GET /api/audit on first show
Pattern: x-if (same as rankings tab — prevents rendering when hidden)
```

### Health Grid Row Click (expand/collapse)

```
Trigger: click any row in the health grid
Action: selectedAuditSlug = slug (toggle: click same row again collapses)
Effect: drill-down panel appears below the clicked row
Data load: GET /api/audit/:slug for full detail (if not already cached)
Animation: none (instant expand/collapse, matching existing dashboard pattern)
```

### Run Audit

```
Trigger: click "Run Audit" button
Action: POST /api/audit/run
Loading: button becomes disabled, shows spinner + "Audit en cours..."
Completion: SSE event { type: 'audit-complete' } triggers data refresh
Error: show red text below button: "Erreur: [message]" for 5 seconds
```

### Generate Patch

```
Trigger: click "Generer patch" in table action column or drill-down panel
Action: POST /api/audit/:slug/patch
Loading: button becomes disabled, shows spinner + "Generation..."
Success: pendingPatch populated, drill-down refreshes to show patch preview
Error: show red text in drill-down: "Erreur de generation: [message]"
```

### Approve Patch

```
Trigger: click "Appliquer le patch" in drill-down
Action: POST /api/audit/:slug/apply
Confirmation: reuse existing _confirmModal pattern — "Appliquer le patch SEO sur [slug] ? Cette action deploiera les modifications sur le site."
Loading: button becomes disabled, shows spinner + "Deploiement..."
Success: pendingPatch cleared, score updates, row refreshes with new score badge
Error: show red text: "Echec du deploiement: [message]. Le fichier original a ete restaure."
```

### Reject Patch

```
Trigger: click "Rejeter" in drill-down
Action: DELETE /api/audit/:slug/patch
Effect: pendingPatch cleared, patch preview disappears, no confirmation needed
```

### SSE Updates

```
Event type: 'audit-complete'
Handler: added to existing evtSource.onmessage switch
Effect: re-fetch GET /api/audit, update health grid data, update avgSeoScore in stats row
```
