Create `autopilot/audit/patch-applier.js` as ESM module with these exports:

**`applySnippet(html, patch)`** -- pure function. Takes original HTML string and patch object `{ snippet, target, action, replaceType }`.
- Uses `cheerio.load(html, { decodeEntities: false })` (Pitfall 1: preserve French chars)
- If `action === 'append'`: `$(patch.target).append(patch.snippet)`
- If `action === 'replace'`: find all `script[type="application/ld+json"]` elements, parse each, if `data['@type'] === patch.replaceType`, replace with `$(el).replaceWith(patch.snippet)`
- Returns `$.html()` (full HTML string)

**`writeBackup(slug, originalHtml, { _stateDir, _writeFile, _mkdirSync } = {})`** -- writes backup to `state/backups/[slug]-[timestamp].html`.
- Timestamp format: `new Date().toISOString().replace(/[:.]/g, '-')` (Pitfall 7: Windows filename safety)
- Creates `state/backups/` dir if not exists via `mkdirSync(backupDir, { recursive: true })`
- Returns `{ backupPath, timestamp }`
- DI: `_stateDir`, `_writeFile` (default writeFileSync), `_mkdirSync` (default mkdirSync)

**`pruneBackups(slug, { keepCount = 3, _stateDir, _readdirSync, _unlinkSync } = {})`** -- keeps last N backups per slug.
- Lists files in `state/backups/` matching `${slug}-*.html`
- Sorts lexicographically (ISO timestamps sort correctly)
- Deletes oldest files until only `keepCount` remain
- DI: `_stateDir`, `_readdirSync`, `_unlinkSync`

**`applyPatch(slug, { _stateDir, _basePath, _sftpClient, _readFile, _writeFile, _mkdirSync, _readdirSync, _unlinkSync, _renameSync, _runAudit } = {})`** -- async, orchestrates the full flow:
1. Read `page-audit.json` from `_stateDir || defaultStateDir`. Get `auditData[slug]`.
2. If no `pendingPatch` on that slug, throw `Error('No pending patch for slug')`
3. Read original HTML from disk: determine file path from page inventory (`buildPageInventory({ _basePath })` then `inventory[slug].filePath`)
4. Apply snippet: call `applySnippet(originalHtml, pendingPatch)`
5. Validate: call `validatePatch(originalHtml, patchedHtml)`. If not valid, return `{ success: false, error: 'Validation failed', failedChecks }`
6. Idempotency check (F4.7 check 5): apply snippet again on patchedHtml, compare result with patchedHtml via `===`. If different, return `{ success: false, error: 'Not idempotent', failedChecks: ['NOT_IDEMPOTENT'] }`
7. Write backup via `writeBackup(slug, originalHtml, ...)`
8. Prune old backups via `pruneBackups(slug, ...)`
9. Atomic local write: write to `filePath + '.tmp'`, then rename to `filePath` (Pitfall 3 pattern from RESEARCH)
10. SFTP deploy single file:
    - Create SFTP client: `_sftpClient || new SftpClient()`
    - Connect with env credentials (same pattern as sftp-deployer.js)
    - Determine remote path from slug: for blog pages `${remote}blog/${slug}.html`, for root pages `${remote}${slug}.html`
    - Upload via temp+rename: `sftp.put(localBuffer, remotePath + '.tmp')`, then `sftp.rename(remotePath + '.tmp', remotePath)`
    - End connection in finally block
11. On SFTP failure: restore original from backup (read backup, write to filePath), log error, return `{ success: false, error: err.message, backupPath }`
12. On success: clear `pendingPatch` from auditData[slug], write updated page-audit.json
13. Re-scan: call `(_runAudit || runAudit)({ slugs: [slug], _basePath, _readFile, _stateDir })` to update scores
14. Write audit event to `state/audit-events.json`: `{ type: 'audit-patch-applied', slug, timestamp, newScore }` (SSE watcher picks this up)
15. Return `{ success: true, backupPath, newScore: rescanResult[slug].score }`

**`rollbackPatch(slug, { _stateDir, _basePath, _sftpClient, _readFile, _writeFile, _readdirSync, _renameSync, _runAudit } = {})`** -- async:
1. Find most recent backup: list `state/backups/` for `${slug}-*.html`, sort, take last
2. If no backup found, throw `Error('No backup found for slug')`
3. Read backup HTML
4. Atomic local write (tmp + rename)
5. SFTP upload single file (same pattern as applyPatch step 10)
6. Re-scan via runAudit
7. Clear `pendingPatch` from page-audit.json if it exists
8. Write audit event: `{ type: 'audit-patch-rolled-back', slug, timestamp }`
9. Return `{ success: true, restoredFrom: backupPath }`

Import: `SftpClient from 'ssh2-sftp-client'`, `{ load } from 'cheerio'`, `pino`, `{ validatePatch } from './patch-validator.js'`, `{ runAudit } from './runner.js'`, `{ buildPageInventory } from './page-inventory.js'`, fs functions from `node:fs`.

**d) Add `POST /api/audit/:slug/patch` route** (for Phase 12 dashboard "Generate Patch" button):
```javascript
apiRouter.post('/audit/:slug/patch', async (req, res) => {
  const { slug } = req.params;
  try {
    // Read current audit data for this slug
    const auditPath = join(STATE_DIR, 'page-audit.json');
    if (!existsSync(auditPath)) return res.status(404).json({ error: 'No audit data' });
    const auditData = JSON.parse(readFileSync(auditPath, 'utf8'));
    const record = auditData[slug];
    if (!record) return res.status(404).json({ error: 'Slug not found in audit' });

    // Build page context for Claude (head excerpt + schema data from config.js)
    const inventory = (await import('../audit/page-inventory.js')).buildPageInventory();
    const entry = inventory[slug];
    if (!entry) return res.status(404).json({ error: 'Page not found in inventory' });
    const html = readFileSync(entry.filePath, 'utf-8');
    const headMatch = html.match(/<head[^>]*>([\s\S]*?)<\/head>/i);
    const headExcerpt = headMatch ? headMatch[1].slice(0, 3000) : '';

    // Load schema data from config.js
    let schemaData = {};
    try {
      const configPath = join(SITE_BASE_PATH, 'assets', 'js', 'config.js');
      const configContent = readFileSync(configPath, 'utf-8');
      const schemaMatch = configContent.match(/schema\s*:\s*(\{[\s\S]*?\n\s*\})/);
      if (schemaMatch) schemaData = { raw: schemaMatch[1].slice(0, 2000) };
    } catch { /* config read failure non-fatal */ }

    const result = await generatePatch(slug, record.issues, { headExcerpt, schemaData });
    if (result.blocked) {
      return res.status(403).json({ error: 'Never-auto-apply guard', reason: result.reason });
    }

    // Store pendingPatch in page-audit.json
    auditData[slug].pendingPatch = {
      snippet: result.snippet,
      target: 'head',
      action: 'append',
      replaceType: null,
      issuesCovered: record.issues.map(i => i.code),
      generatedAt: new Date().toISOString(),
      estimatedCostUsd: result.usage ? ((result.usage.input_tokens + result.usage.output_tokens) / 1000000 * 15) : 0,
    };
    writeFileSync(auditPath, JSON.stringify(auditData, null, 2), 'utf-8');

    res.json({ ok: true, snippet: result.snippet, issuesCovered: auditData[slug].pendingPatch.issuesCovered });
  } catch (err) {
    logger.error({ slug, err: err.message }, 'Patch generation failed');
    res.status(500).json({ error: err.message });
  }
});
```
