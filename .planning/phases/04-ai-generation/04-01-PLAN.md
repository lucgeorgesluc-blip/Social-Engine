**4. dashboard/lib/ai.js** — Claude API wrapper. Uses CJS (`require`). Exports `generatePost`, `getMonthlyUsage`, `getFrequentObjections`.

```javascript
const Anthropic = require('@anthropic-ai/sdk').default;
const { query } = require('./db');
const { buildPrompt } = require('./prompts');

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY });

function currentMonthKey() {
  const d = new Date();
  return `${d.getFullYear()}-${String(d.getMonth() + 1).padStart(2, '0')}`;
}
```

`getMonthlyUsage()` — returns `{ count, limit: 100, warning: 50 }`:
```sql
SELECT COUNT(*) as count FROM ai_generations WHERE month_key = $1
```
Pass `currentMonthKey()`.

`generatePost(type, context)` — the main generation function:
1. Call `getMonthlyUsage()`. If `count >= 100`, throw `Error('LIMIT_REACHED')`. 
2. Call `buildPrompt(type, context)` to get `{ system, user }`.
3. Call `client.messages.create({ model: 'claude-haiku-4-5-20251001', max_tokens: 1024, system, messages: [{ role: 'user', content: user }] })`.
4. Extract text: `const text = message.content[0].text`.
5. Insert tracking row: `INSERT INTO ai_generations (month_key, post_type, tokens_used, generated_text) VALUES ($1, $2, $3, $4)` with `currentMonthKey()`, `type`, `message.usage.output_tokens`, `text`.
6. Get updated usage with `getMonthlyUsage()`.
7. Return `{ text, usage }` where usage includes count, limit, warning, and `isWarning: usage.count >= 50`.

`getFrequentObjections()` — returns array of `{ objection_type, frequency }`:
```sql
SELECT objection_type, COUNT(*) as frequency
FROM comments
WHERE objection_type IS NOT NULL
  AND objection_type != ''
  AND created_at > NOW() - INTERVAL '90 days'
GROUP BY objection_type
HAVING COUNT(*) >= 3
ORDER BY frequency DESC
```

All functions wrapped in try/catch — rethrow with descriptive messages.

**1. dashboard/routes/ai.js** — Express router for AI endpoints. All routes require auth.

```javascript
const express = require('express');
const router = express.Router();
const { isAuthenticated } = require('./auth');
const { generatePost, getMonthlyUsage, getFrequentObjections } = require('../lib/ai');
const { POST_TYPES } = require('../lib/prompts');
```

All routes use `isAuthenticated` middleware.

`GET /` (serves as `/dashboard/ai` via mount) — render `ai` view:
- Call `getMonthlyUsage()` to get current usage.
- Call `getFrequentObjections()` to get suggestions.
- Render `ai.ejs` with `{ postTypes: POST_TYPES, usage, suggestions, generated: null, error: null, selectedType: null, context: '' }`.

`POST /generate` (serves as `/dashboard/ai/generate`) — generate a post:
- Extract `type` and `context` from `req.body`.
- Validate: if `type` is not in POST_TYPES ids, return render with `error: 'Type de post invalide'`.
- Try: call `generatePost(type, context)`. Render `ai.ejs` with `{ generated: result.text, usage: result.usage, postTypes: POST_TYPES, suggestions: await getFrequentObjections(), error: null, selectedType: type, context }`.
- Catch: if `err.message === 'LIMIT_REACHED'`, render with `error: 'Limite mensuelle atteinte (100 generations). Reessayez le mois prochain.'`. Otherwise render with `error: 'Erreur de generation: ' + err.message`.

`GET /usage` (serves as `/dashboard/ai/usage`) — JSON endpoint for AJAX:
- Call `getMonthlyUsage()`, return `res.json(usage)`.
