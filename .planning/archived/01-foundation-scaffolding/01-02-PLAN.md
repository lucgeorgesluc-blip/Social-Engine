1. Read GOOGLE_AI_API_KEY from the user's `.env` file (or ask user to provide it if not set). The key must be valid — created at https://aistudio.google.com/apikey.

2. Call the Google AI models list endpoint to find image-capable models. Use Node.js fetch (not curl, to avoid Windows shell issues):
```javascript
// Run from autopilot/ directory
// node -e script or create a temporary verify-gemini.js

import { config } from 'dotenv';
config();

const key = process.env.GOOGLE_AI_API_KEY;
if (!key) { console.error('Set GOOGLE_AI_API_KEY in .env'); process.exit(1); }

const res = await fetch(`https://generativelanguage.googleapis.com/v1beta/models?key=${key}`);
const data = await res.json();

// Filter for image generation capable models
const imageModels = data.models.filter(m =>
  m.name.includes('imagen') ||
  m.name.includes('image') ||
  (m.supportedGenerationMethods || []).includes('generateImages')
);

console.log('Image-capable models:');
imageModels.forEach(m => console.log(`  ${m.name} — ${m.description?.slice(0, 100)}`));
```

3. From the output, identify the correct model name for image generation. Candidates in priority order:
   - `gemini-2.0-flash-preview-image-generation` (multimodal generation including images)
   - `imagen-3.0-generate-002` (dedicated image model)
   - Any other model with image generation support

4. Write the verified model name (just the string, e.g. `models/imagen-3.0-generate-002`) as a single line to `autopilot/config/gemini-model.txt`. No trailing newline, no comments, just the model identifier string.

```javascript
import { describe, it } from 'node:test';
import assert from 'node:assert/strict';
import { readFileSync } from 'node:fs';
import { join, dirname } from 'node:path';
import { fileURLToPath } from 'node:url';

const __dirname = dirname(fileURLToPath(import.meta.url));

describe('gemini-model.txt', () => {
  it('exists and contains a model name', () => {
    const content = readFileSync(join(__dirname, '..', 'config', 'gemini-model.txt'), 'utf8').trim();
    assert.ok(content.length > 5, 'Model name should be more than 5 chars');

      content.includes('imagen') || content.includes('gemini') || content.includes('image'),
      `Model name should contain imagen, gemini, or image — got: ${content}`
    );
  });
});
```
