```sql
CREATE TABLE IF NOT EXISTS fb_tokens (
  id SERIAL PRIMARY KEY,

  access_token TEXT NOT NULL,
  page_id VARCHAR(50) NOT NULL,
  expires_at TIMESTAMPTZ,

  updated_at TIMESTAMPTZ DEFAULT NOW()

  updated_at TIMESTAMPTZ DEFAULT NOW()

```

**dashboard/lib/fb-token.js** — Token service module (CJS):

Export these functions:
1. `getPageToken()` — `SELECT * FROM fb_tokens WHERE token_type = 'page_access' ORDER BY created_at DESC LIMIT 1`. Return row or null.
2. `storePageToken({ access_token, page_id, expires_at, scopes })` — DELETE all existing page_access tokens, then INSERT new one. Returns the inserted row.
3. `checkTokenHealth(debugResponse)` — Pure function (no DB). Takes the debug_token API response object (or null). Returns `{ status, color, message, daysLeft }` per the banner logic from research:
   - `null` input -> `{status: 'missing', color: 'red', message: 'Aucun token Facebook — Connecter'}`
   - `is_valid === false` -> `{status: 'expired', color: 'red', message: 'Token Facebook expiré — Reconnecter'}`
   - `is_valid === true` AND expires_at <= 7 days from now -> `{status: 'expiring', color: 'amber', message: 'Token expire dans X jours — Renouveler', daysLeft: X}` where X = Math.ceil((expires_at - now) / 86400000)
   - `is_valid === true` AND expires_at > 7 days -> `{status: 'valid', color: 'green', message: 'Token Facebook valide'}`
4. `debugToken(accessToken)` — Calls `https://graph.facebook.com/debug_token?input_token=${accessToken}&access_token=${process.env.FB_APP_ID}|${process.env.FB_APP_SECRET}` using Node 20 `fetch()`. Returns parsed JSON `.data` field. On error returns null.
5. `exchangeCodeForToken(code, redirectUri)` — Exchanges OAuth code for short-lived token, then long-lived token, then derives page token. Three sequential fetch calls per research Section 2. Returns `{ access_token, page_id, expires_at }` or throws.

All DB functions use `require('./db').query`.

**dashboard/routes/fb.js** — Facebook routes (CJS, all auth-protected except callback):

```javascript
const express = require('express');
const router = express.Router();
const { isAuthenticated } = require('./auth');
const { getPageToken, checkTokenHealth, debugToken, exchangeCodeForToken, storePageToken } = require('../lib/fb-token');
```

Routes:

1. `GET /dashboard/fb/auth` (isAuthenticated) — Redirect to Facebook OAuth URL:
   `https://www.facebook.com/v21.0/dialog/oauth?client_id=${process.env.FB_APP_ID}&redirect_uri=${encodeURIComponent(redirectUri)}&scope=pages_read_engagement,pages_read_user_content,read_insights,pages_manage_posts&state=${csrfState}`
   - `redirectUri` = `${req.protocol}://${req.get('host')}/dashboard/fb/callback`
   - `csrfState` = random 32-char hex string (use `require('crypto').randomBytes(16).toString('hex')`)
   - Store csrfState in `req.session.fbOAuthState`

2. `GET /dashboard/fb/callback` (isAuthenticated) — Handle OAuth callback:
   - Validate `req.query.state === req.session.fbOAuthState` (CSRF protection). If mismatch, redirect to `/` with error flash.
   - Call `exchangeCodeForToken(req.query.code, redirectUri)` where redirectUri matches the one used in /auth
   - Call `storePageToken(result)` to save token
   - Redirect to `/` (dashboard homepage)
   - On error: redirect to `/` (token will show as missing in banner)

3. `GET /dashboard/fb/status` (isAuthenticated) — AJAX endpoint for token health:
   - `const token = await getPageToken()`
   - If token: `const debug = await debugToken(token.access_token)`, then `const health = checkTokenHealth(debug)`
   - If no token: `const health = checkTokenHealth(null)`
   - Return JSON: `res.json(health)`

Export `router`.

**dashboard/views/partials/fb-banner.ejs** — Token health banner partial:
```html
<div id="token-health-banner" class="mb-4 px-4 py-3 rounded-lg flex items-center justify-between text-sm">
  <span id="token-health-message">Chargement...</span>
  <a id="token-health-action" href="/dashboard/fb/auth" class="hidden font-medium underline">Connecter</a>
</div>
<script>
  fetch('/dashboard/fb/status')
    .then(r => r.json())
    .then(data => {
      const banner = document.getElementById('token-health-banner');
      const msg = document.getElementById('token-health-message');
      const action = document.getElementById('token-health-action');
      msg.textContent = data.message;
      banner.classList.remove('bg-green-100', 'text-green-800', 'bg-amber-100', 'text-amber-800', 'bg-red-100', 'text-red-800');
      if (data.color === 'green') {
        banner.classList.add('bg-green-100', 'text-green-800');
      } else if (data.color === 'amber') {
        banner.classList.add('bg-amber-100', 'text-amber-800');
        action.textContent = 'Renouveler';
        action.classList.remove('hidden');
      } else {
        banner.classList.add('bg-red-100', 'text-red-800');
        action.textContent = data.status === 'missing' ? 'Connecter' : 'Reconnecter';
        action.classList.remove('hidden');
      }
    })
    .catch(() => {
      document.getElementById('token-health-banner').classList.add('bg-gray-100', 'text-gray-600');
      document.getElementById('token-health-message').textContent = 'Statut Facebook indisponible';
    });
</script>
```
