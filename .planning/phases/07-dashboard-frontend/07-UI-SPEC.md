# Phase 7 — UI Design Contract: Dashboard Frontend

> Visual and interaction contract for the Mac-app dark dashboard.
> Generated by gsd-ui-researcher. Consumed by gsd-planner, gsd-executor, gsd-ui-auditor.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (CDN-only, no component library) |
| Preset | not applicable |
| Component library | none — hand-rolled with Tailwind CSS v3 utility classes |
| Icon library | Heroicons v2 inline SVG (no external dependency — paste SVG paths directly) |
| Font | system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif |

**No shadcn.** This project uses no build step and no React. All components are bespoke HTML+CSS patterns using Tailwind utility classes.

## CDN Script Load Order

Scripts must appear in `<head>` in this exact order:

```html
<!-- 1. Tailwind config BEFORE cdn.tailwindcss.com -->
<script>
  tailwind.config = {
    darkMode: 'class',
    theme: {
      extend: {
        colors: {
          navy: {
            950: '#0d1117',
            900: '#161b22',
            800: '#1c2128',
            700: '#21262d'
          },
          accent: {
            DEFAULT: '#3b82f6',
            hover:   '#1d4ed8'
          }
        },
        borderRadius: {
          card: '12px'
        }
      }
    }
  }
</script>

<!-- 2. Tailwind CSS v3 Play CDN -->
<script src="https://cdn.tailwindcss.com"></script>

<!-- 3. Alpine.js v3.15.9 — defer is MANDATORY -->
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.15.9/dist/cdn.min.js"></script>

<!-- 4. Chart.js (rankings tab only — load on index.html, not login.html) -->
<script src="https://cdn.jsdelivr.net/npm/chart.js@4.5.1/dist/chart.umd.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/date-fns@2/index.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/chartjs-adapter-date-fns@3.0.0/dist/chartjs-adapter-date-fns.bundle.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/chartjs-plugin-annotation@3.1.0/dist/chartjs-plugin-annotation.min.js"></script>

<!-- 5. D3 v7 full bundle (link tree tab only — load on index.html) -->
<script src="https://cdn.jsdelivr.net/npm/d3@7.9.0/dist/d3.min.js"></script>
```

**Critical:** `login.html` does NOT load Chart.js or D3 — only Tailwind + Alpine.

## Spacing Scale

All values are multiples of 4. Mapped to Tailwind v3 numeric scale.

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| xs | 4px | `p-1` / `gap-1` | Icon-to-label gaps, badge inner padding |
| sm | 8px | `p-2` / `gap-2` | Compact row padding, button icon margins |
| md | 16px | `p-4` / `gap-4` | Default cell padding, card inner spacing |
| lg | 24px | `p-6` / `gap-6` | Card padding, section padding |
| xl | 32px | `p-8` / `gap-8` | Layout gaps between cards |
| 2xl | 48px | `py-12` | Major section breaks |
| 3xl | 64px | `py-16` | Page-level top/bottom padding |

**Exceptions:**
- Sidebar width: 240px (fixed, not on 8-pt scale — required for text legibility of nav labels)
- Touch targets: minimum 44px height on all interactive elements (WCAG 2.1 AA)
- Stepper circle: 32px diameter (`w-8 h-8`)
- Stats row stat number: top margin 4px between label and value only

## Accessibility Requirements

### Keyboard Navigation

- All interactive elements (`button`, `input`, `a`) are natively focusable — no `tabindex` hacks needed
- Sidebar nav buttons: `Tab` moves through each, `Enter` activates
- Table action buttons: accessible via `Tab` within each row
- Modal: on open, focus moves to first button; `Escape` closes; focus returns to trigger on close
- Chart.js: provide `aria-label="Graphique des classements GSC"` on `<canvas>`
- D3 tree: `<svg aria-label="Arbre de maillage interne" role="img">` — complex interaction, no full keyboard support required (internal tool)

### ARIA Labels

| Element | ARIA |
|---------|------|
| Sidebar `<nav>` | `aria-label="Navigation principale"` |
| Pending badge span | `aria-label="Articles en attente d'approbation : {count}"` |
| SSE status dot | `aria-live="polite"` on parent container |
| Chart canvas | `aria-label="Graphique des classements Google Search Console"` |
| Stats cards | Each `<p>` stat label acts as accessible name for its value |
| Delete modal | `role="dialog"`, `aria-modal="true"`, `aria-labelledby="modalTitle"` |
| Approve button | `aria-label="Approuver et déployer l'article : {title}"` |
| Activity feed list | `<ul aria-label="Journal d'activité">` |

### Color Contrast

All text/background combinations must meet WCAG 2.1 AA (4.5:1 for normal text, 3:1 for large text):
- `#e6edf3` on `#1c2128`: ~11:1 — PASS
- `#7d8590` on `#1c2128`: ~4.6:1 — PASS (body label on card)
- `#e6edf3` on `#0d1117`: ~13:1 — PASS
- `#0d1117` on `#22c55e` (badge): ~8.5:1 — PASS
- `#0d1117` on `#eab308` (badge): ~9.7:1 — PASS
- White on `#3b82f6` (blue button): ~3.0:1 — PASS for large/bold text (badges, buttons ≥ 14px bold)

### Focus Ring

All buttons and inputs: `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-[#3b82f6] focus-visible:ring-offset-2 focus-visible:ring-offset-[#0d1117]`
