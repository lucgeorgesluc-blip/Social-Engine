    **Step 2: Create `autopilot/audit/cannibalization.js`**

    ESM module with three named exports:

    `normalizeTokens(text)`:
    - `text.normalize('NFD').replace(/[\u0300-\u036f]/g, '')` to strip accents
    - `.toLowerCase()`
    - `.split(/[\s\-:,|.!?'"()\[\]\/]+/)` to tokenize on whitespace + punctuation
    - Also split on apostrophes: replace `'` and `'` with space before splitting
    - `.filter(t => t.length > 1)` to drop single chars
    - `.filter(t => !FR_STOPWORDS.has(t))` to remove stopwords
    - Returns `string[]`

    `jaccard(tokensA, tokensB)`:
    - Create `Set` from each array
    - Count intersection size by iterating setA, checking setB.has()
    - Union size = setA.size + setB.size - intersectionSize
    - Return 0 if union is 0, else intersectionSize / unionSize
    - Returns `number` (0-1)

    `detectCannibalization(pages, contentMapEntries)`:
    - `pages` param: `Array<{ slug, title, h1Text }>` — pre-extracted page data (DI, no I/O in this module)
    - `contentMapEntries` param: `Array<{ slug, title, cluster_id }>` — from content-map.yaml
    - Build a `Map<slug, { tokens, cluster_id }>` by combining page title + h1Text into a single token set per slug, and looking up cluster_id from contentMapEntries
    - **First pass (same-cluster):** Group pages by cluster_id. For each cluster with 2+ pages, compute Jaccard for all pairs. If Jaccard >= 0.15, add to `sameClusterPairs[]`
    - **Second pass (cross-cluster):** For all pairs NOT in the same cluster, only compute Jaccard if they share >= 2 raw tokens (pre-filter optimization). If Jaccard >= 0.15, add to `crossClusterPairs[]`
    - Each pair object: `{ slugA, slugB, similarity: number, sharedTokens: string[], severity: 'critical'|'medium'|'low', sameCluster: boolean, clusterId: string|null }`
    - Severity thresholds: >0.85 = "critical", 0.60-0.85 = "medium", 0.15-0.60 = "low"
    - Return `[...sameClusterPairs, ...crossClusterPairs]` sorted by similarity descending within each group
    - No I/O, no imports except FR_STOPWORDS — pure function
