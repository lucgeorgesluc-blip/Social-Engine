
    Parameters:
    - `slug` — article slug (e.g., "auriculotherapie-arret-tabac")
    - `articlePath` — absolute local path to blog/[slug].html
    - `imagePath` — absolute local path to image, or null if image generation failed
    - `basePath` — SITE_BASE_PATH (e.g., "E:/Site CL")

    - `_sftpClient` — injected SFTP client for tests

    File mapping (local -> remote):
    ```javascript
    const files = [
      { local: articlePath, remote: `${remoteBasePath}blog/${slug}.html` },
      // imagePath only if not null:
      { local: imagePath, remote: `${remoteBasePath}assets/images/blog/${slug}.webp` },
      { local: join(basePath, 'assets/js/config.js'), remote: `${remoteBasePath}assets/js/config.js` },
      { local: join(basePath, 'assets/js/config.min.js'), remote: `${remoteBasePath}assets/js/config.min.js` },
      { local: join(basePath, 'sitemap.xml'), remote: `${remoteBasePath}sitemap.xml` },
    ];
    ```

    Filter out null imagePath entries before deploying.

    Atomic deploy pattern:
    1. `sftp.connect({ host, port, username, password })` — read from process.env
    2. For each file: `await pRetry(() => sftp.put(local, remote + '.tmp'), { retries: 1 })`
    3. After ALL puts succeed: `for (const f of files) await sftp.rename(remote + '.tmp', remote)`
    4. `sftp.end()` in finally block

    Use pino logger (import pino, create logger with name 'sftp-deployer').

    Return `{ success: true, filesDeployed: files.length }` on success.
    Throw on failure (caller handles error reporting).

    Also add `SFTP_REMOTE_PATH` to .env.example with comment `# Remote web root on IONOS (e.g., / or /html/)`.

    Function signature:
    ```javascript
    export async function pingGsc({ articleUrl, _googleAuth, _searchConsole }) {
    ```

    Parameters:
    - `articleUrl` — full URL like `https://www.magnetiseuse-lacoste-corinne.fr/blog/slug.html`
    - `_googleAuth` — injected auth client for tests
    - `_searchConsole` — injected searchconsole client for tests

    Implementation:
    1. Check `process.env.GSC_SERVICE_ACCOUNT_PATH` — if missing, return `{ status: "skipped", reason: "missing credentials" }`
    2. Create auth: `const auth = _googleAuth || new google.auth.GoogleAuth({ keyFile: process.env.GSC_SERVICE_ACCOUNT_PATH, scopes: ['https://www.googleapis.com/auth/webmasters'] })`
    3. Create client: `const searchConsole = _searchConsole || google.searchconsole({ version: 'v1', auth })`
    4. Call: `searchConsole.urlInspection.index.inspect({ requestBody: { inspectionUrl: articleUrl, siteUrl: 'https://www.magnetiseuse-lacoste-corinne.fr/' } })`
    5. Parse response: check `inspectionResult.indexStatusResult.coverageState`
       - Contains "indexed" (case-insensitive) -> `{ status: "already_indexed" }`
       - Otherwise -> `{ status: "submitted" }`

    The siteUrl `https://www.magnetiseuse-lacoste-corinne.fr/` comes from a const at module top (not hardcoded inside the function body) so it can be found and changed if needed.
